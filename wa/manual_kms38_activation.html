KMS38 ​

این صفحه برای کاربرانی در نظر گرفته شده است که به هر دلیلی نمی‌خواهند از اسکریپت استفاده کنند و ترجیح می‌دهند فرآیند فعال‌سازی KMS38 را خودشان انجام دهند. اگر به جای آن می‌خواهید از ابزاری برای این کار استفاده کنید، لطفاً اینجا را بررسی کنید.

نکته

فعال‌سازی KMS38 فقط در نسخه‌های کلاینت و سرور ویندوز 10/11 پشتیبانی می‌شود.

فعال‌سازی دستی ویندوز ​

برای فعال‌سازی دستی ویندوز، این مراحل را دنبال کنید:

  • مرحله 1: Windows Powershell را به عنوان مدیر باز کنید و دستورات زیر را وارد کنید.

برای اجرای Powershell

برای مشاهده جزئیات اینجا کلیک کنید

گزینه الف: استفاده از نوار جستجو توصیه می‌شود ​

  1. روی دکمه «Start» ویندوز یا نماد «Search» در نوار وظیفه کلیک کنید.
  2. powershell را تایپ کنید.
  3. Run as administrator را انتخاب کنید.

گزینه ب: استفاده از کادر Run ​

  1. کادر محاوره‌ای Run را باز کنید: کلیدهای Win (⊞) + R را همزمان فشار دهید.
  2. powershell را تایپ کنید.
  3. برای اجرای به عنوان مدیر، Ctrl + Shift + Enter را فشار دهید.
  4. اگر توسط کنترل حساب کاربری درخواست شد، روی Yes کلیک کنید.

گزینه ج: استفاده از منوی Power User ​

  1. روی «Start menu» راست کلیک کنید (یا Win (⊞) + X را فشار دهید).
  2. «Windows Terminal (Admin)» را در ویندوز 11 یا «Windows PowerShell (Admin)» را در ویندوز 10 انتخاب کنید.

یا می‌توانید از این لینک برای دیدن 10 روش برای اجرای PowerShell و این لینک برای اجرای CMD در ویندوز خود بازدید کنید.

  • مرحله 2: کلید عمومی را که با نسخه ویندوز شما مطابقت دارد نصب کنید:
reg
slmgr /ipk <key>
1

(برای کلید مناسب به جدول زیر مراجعه کنید.)

  • مرحله 3: فایل تیکت KMS38 را دانلود کنید.
  • مرحله 4: فایل تیکت دانلود شده را در ریشه درایو C: کپی کنید.
  • مرحله 5: با اجرای دستورات زیر، تیکت را به یک لایسنس منتقل کرده و ویندوز را فعال کنید:
reg
clipup -v -o -altto C:\
1
  • در نهایت پس از چند ثانیه انتظار، ویندوز باید فعال شود.

نکات

  • برای نسخه‌های Windows Server Cor/Acor، سیستم فایل clipup.exe را ندارد. برای فعال‌سازی آن با استفاده از KMS38، فایل گمشده ClipUp.exe را از این لینک دانلود کنید. File: ClipUp.exeSHA-256: 0d6e9f6bbd0321eda149658d96040cb4f79e0bd93ba60061f25b28fecbf4d4ef این فایل به صورت دیجیتالی امضا شده و قابل تأیید است. همچنین می‌توانید این فایل را از ISO رسمی Windows Server 2016 x64 RTM دریافت کنید. فایل ClipUp.exe را در پوشه C:\Windows\System32 قرار دهید و فرآیند فعال‌سازی KMS38 را انجام دهید. پس از اتمام فعال‌سازی، می‌توانید فایل را حذف کنید.
