KMS38 ‚Äã

This page is intended for users who do not want to use the script for any reason and would rather perform the KMS38 activation process themselves. If you'd like to use a tool for this instead, please check here.

Note

KMS38 activation is only supported on Windows 10/11 Client and Server editions.

Manually Activate Windows ‚Äã

To manually activate Windows, follow these steps:

  • step 1: Open Windows Powershell as administrator and enter the below commands

To run Powershell

Click here to see details

Option A: Using Search Bar Recommend ‚Äã

  1. Click on windows "Start" button or "Search" icon in the taskbar
  2. Type powershell
  3. Select Run as administrator

Option B: Using Run Box ‚Äã

  1. Open the Run dialog box: Press Win (‚äû) + R keys simultaneously
  2. Type powershell
  3. Press Ctrl + Shift + Enter to run as administrator
  4. If prompted by User Account Control, click Yes

Option C: Using Power User Menu ‚Äã

  1. Right-click on the "Start menu" (or press Win (‚äû) + X)
  2. Select "Windows Terminal (Admin)" on Windows 11 or "Windows PowerShell (Admin)" on Windows 10

Or You can visit this Link to see 10 ways to run PowerShell And this link For Running CMD in your windows.

  • step 2: Install the generic key which matches your Windows edition and version:
reg
slmgr /ipk <key>
1

(Refer to the table below for the appropriate key.)

  • Step 3: Download the KMS38 ticket file.
  • Step 4: Copy the downloaded ticket file to the root of the C: drive.
  • Step 5: Migrate the ticket to a license and activate Windows by running the commands:
reg
clipup -v -o -altto C:\
1
  • Finally After waiting for a few seconds, Windows should be activated.

Notes

  • For Windows Server Cor/Acor editions, the system does not have the clipup.exe file. To activate it using KMS38, download the missing ClipUp.exe file from this link. File: ClipUp.exeSHA-256: 0d6e9f6bbd0321eda149658d96040cb4f79e0bd93ba60061f25b28fecbf4d4ef The file is digitally signed and verifiable. You can also obtain this file from the official Windows Server 2016 x64 RTM ISO. Place the ClipUp.exe file in the C:\Windows\System32 folder and perform the KMS38 activation process. Once the activation is complete, you can remove the file.
